# 🧠 НейроЗеркало

Интерактивный **3D-атлас мозга ваших состояний**. Опишите словами, что вы
чувствуете, испытываете или как прошёл день — приложение подсветит на вращаемой
модели мозга ответственные структуры, покажет потоки нейромедиаторов, объяснит
механизм простым научным языком и даст доказательные (в т.ч. самые свежие и
нестандартные) советы со ссылками на исследования.

Главная цель — помочь человеку лучше познакомиться с собой и своей
нейробиологией.

## Как запустить

Работает в браузере и **офлайн** (Three.js вложен в `lib/`).

**Просто откройте файл:** дважды кликните `index.html`.

**Или локальный сервер (для PWA-установки):**
```bash
cd neuro-mirror
python3 server.py
```
→ откройте http://localhost:4599

**Установка на телефон/компьютер (PWA):** нажмите кнопку **⤓ Установить**
вверху — она сама покажет нужные шаги:
- **iPhone/iPad:** откройте адрес в **Safari** → «Поделиться» → «На экран
  “Домой”» → «Добавить».
- **Android/компьютер (Chrome/Edge):** значок установки в адресной строке или
  «…» → «Установить приложение».

Для установки нужен доступ по адресу `http(s)://…` (через `server.py` или
хостинг), а не открытие файла напрямую.

## Возможности

**Мозг**
- 3D-модель, вращаемая мышью/пальцем; ракурсы 3/4 · сбоку · спереди · сверху.
- Подсветка ключевых (🔴) и сопутствующих (🟡) структур с подписями.
- Потоки нейромедиаторов (дофамин, кортизол, норадреналин, серотонин) + легенда.
- Смешанные состояния: связь «конфликта» между спорящими зонами.
- Режим «до/после»: 🟢 как меняется мозг после применения совета.

**Анализ**
- Распознаёт эмоции, симптомы **и образ жизни** («мало спал», «много ел»,
  «ни с кем не общалась» → вялость и её причины).
- Свежие и нестандартные техники со **ссылками на исследования** и уровнями
  достоверности (доказано / новые данные / предварительно).
- Научно изучаемые **добавки** под ситуацию (с дисклеймером).

**Разделы**
- **Исследовать** — 3 режима: **Схема** (дружелюбная кликабельная карта головы:
  нажми структуру → простыми словами её функция + ассоциация), **Путешествие**
  (послойно: доли → эмоции → мотивация → нейромедиаторы, с мини-тестами) и
  **Атлас** (свободный клик по структурам).
- **Программы** — пошаговые практики «перепрограммирования» мозга с трекером и
  сериями дней: дофаминовая перезагрузка, как правильно скучать, перепрошивка
  привычки. С научным обоснованием и ссылками.
- **Открытия** — лента свежих находок нейронауки с метками достоверности
  (✅ проверено · 🧪 новые данные · ❓ не подтверждено) и ссылками на источники;
  готова к авто-обновлению из внешней ленты или через ИИ-хук.
- **Библиотека** — все состояния, которые понимает приложение, с поиском.
- **Курс** — мини-уроки о каждой структуре + факты, с прогрессом.
- **Викторина** — «угадай зону мозга», очки и объяснения.
- **Дневник** — записи + **графики динамики** (настроение за 2 недели, дни
  недели, корреляция триггеров), выявление закономерностей, **таймлайн
  триггеров** (сон, спорт, кофе…), экспорт **CSV/PDF**, **шеринг**-карточка,
  зашифрованная **резервная копия** для переноса между устройствами.

**Прочее**
- **Мультиязычность** (RU/EN, переключатель вверху).
- **PWA** (офлайн + иконка на экране).

## Движок распознавания (гибридный)

Сейчас — **встроенная база** по ключевым словам (офлайн, без ключей). Код готов
к ИИ: определите `window.NEURO_AI_PROVIDER(text)` (async → `[{id, score}]`).
Если задана — используется она, иначе локальный движок (`analyzeText` в `app.js`).

## Про ссылки на исследования

Ссылки ведут на DOI конкретных работ либо на поиск в PubMed по теме. Каждый
совет помечен уровнем достоверности. Это образовательная подборка, а не
клинические рекомендации.

## Лента «Открытия» и авто-обновление

Стартовый набор находок лежит в `updates.js` (`DISCOVERIES_SEED`), каждая с
меткой достоверности и источником. Приложение может подтягивать новые:

- **Внешняя лента:** задайте `window.NEURO_UPDATES_URL` = URL JSON-массива
  находок того же формата — новые пункты подмешаются и закэшируются.
- **ИИ-хук:** реализуйте `window.NEURO_RESEARCH_PROVIDER(topic)` (async →
  массив находок), чтобы модель приносила свежие данные.

Полностью автоматический сбор «всех мировых данных» намеренно не делается: без
проверки это риск дезинформации. Поэтому всё непроверенное честно помечается
как **«ещё не подтверждено»**, а поток обновлений идёт через курируемую ленту
или ИИ-хук, которые вы контролируете.

## ⚠️ Важно

Это **образовательный инструмент**, а не диагностика. Он показывает упрощённую
нейробиологию и не заменяет консультацию врача или психотерапевта. Советы по
добавкам — не назначение; обсуждайте их с врачом. При тяжёлых, длительных
состояниях или мыслях о причинении себе вреда — обратитесь за профессиональной
помощью.
